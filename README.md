# 🔮 personal-ai-engineer-bootcamp

A powerful roadmap / planner for a 10-week self-taught AI Engineer bootcamp — part dashboard, part spellbook. ✨

**🌐 Live site:** https://mycrazyideas.github.io/personal-ai-engineer-bootcamp/

## 🧙 What's inside

- 📊 **Dashboard** — overall progress, streaks, study hours, and a randomized inspirational quote
- 🎵 **Ambient music player** — magical background music that keeps playing as you navigate
- 📅 **Weekly planner** — day-by-day tasks and an editable hour-by-hour schedule
- 📚 **Resource library** — curated + custom learning resources per week
- 🧪 **Build-from-scratch challenges** for hands-on practice
- 📜 **Research paper tracker**
- 🗝️ **Interview prep**
- 🖋️ **Learning journal**
- 🃏 **Flashcard reminders**
- 🏆 **Portfolio tracker**
- 🌿 **Lifestyle & sleep tracking**
- 🌗 **Light / dark mode** with full-screen themed backgrounds
- 💾 **Local storage persistence** — all progress saved right in your browser
- 📱 **Fully responsive** — works great on desktop and mobile
- ⬇️⬆️ **Export / import** your progress as JSON

## 🚀 Getting started

Just open `index.html` in your browser — no build step, no dependencies. 🪄

To run it locally with your own assets, drop these files next to `index.html`:
- `light.jpg` — light mode background
- `dark.png` — dark mode background
- `music.mp3` — ambient study music

## 🛠️ Tech

Pure HTML, CSS, and JavaScript. No frameworks, no build tools.
